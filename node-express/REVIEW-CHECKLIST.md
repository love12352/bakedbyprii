# bakedbyPrii — Review & What's Left Before Launch

A plain-English summary of the website so you can see what's done and what still needs your input.

---

## 👀 How to view the site

Pick whichever is easiest:

1. **Quick visual look (no setup):** double-click `public/index.html` to open it in a web browser.
   You'll see the whole design, menu, sections and legal pages. *(Placing a live order won't work this
   way — that needs the server running, see option 2.)*
2. **The full working site:** in the project folder run `npm install` then `npm start`, and open
   **http://localhost:3000**. Orders, the database and the admin dashboard (**/admin**) all work.
3. **Share a clickable link:** the site can be deployed to a free host (Render / Railway) to get a URL
   you can text to anyone. Ask and this can be set up.

---

## ✅ What's already built

- A polished one-page storefront (Cakes / Cookies / Cupcakes) with animated, photo-based design.
- **Menu**, **Custom Orders**, and **Reviews** sections.
- A full **checkout**: collection from home (free, anyone) or **delivery within Didcot (OX11)**,
  free over £20 otherwise £2.50.
- Payment options: cash, bank transfer, card link, PayPal.
- A real **order system** — orders are saved to a database and show up in a private **admin dashboard**
  where you can mark them New → Confirmed → Completed.
- UK **legal pages**: Allergens, Privacy, Cookies, Terms, Delivery & Returns, plus a cookie banner.
